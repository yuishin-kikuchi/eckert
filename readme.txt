■ソフトウェアの説明（概略）
コンソール動作の、コマンド入力式の電卓アプリケーションです。
操作は逆ポーランド方式となっています。
詳しい操作は、ディレクトリdocのPDFをご覧ください。
<====================================================================>
■インストール方法
exeファイルをお好きなディレクトリにコピーしてお使いください。
専用ディレクトリの作成をおすすめします。
  eckert86.exe  -- 32ビット版
  eckert64.exe  -- 64ビット版

■アンインストール方法
exeファイルを削除するだけです。

■起動方法
exeファイルをダブルクリックして起動します。

■終了方法
「q」と入力し、エンターキーを押すと終了します。
